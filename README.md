# Butts-Abigail-a3-2D-Game
RULES OF THE GAME:
1. The goal of the game is to reach the yellow box at the end.
2. One of the obstacles is the red boxes around the map, they will restart you to the starting position if touched.
3. Input 'W' will make the player jump.
4. Input 'A' will make the player go left.
5. Input 'D' will make the player go right.
